# Global AI & Data Science Salary Analysis

## Overview
This project analyzes global salary trends in AI and data science roles using Python, SQL, and statistical analysis techniques. The goal was to identify compensation patterns across industries, experience levels, and geographic regions.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- SQL
- Quarto

## Key Objectives
- Clean and preprocess salary data
- Analyze salary distributions
- Identify trends across countries and job titles
- Visualize compensation patterns

## Key Findings
- Senior-level roles consistently showed the highest salary growth
- Geographic region strongly influenced compensation
- Remote positions displayed higher salary variability

- `FinalProject.qmd` → Main analysis report
- `FinalProject.pdf` → Final rendered report
